## 二、文章分类标准

### 分类维度定义

| 分类ID | 分类名称 | 特征描述 | 内容定位 |
|:---:|:---|:---|:---|
| L1 | 入门层 | 系统概览、快速入门、术语表 | 面向新用户的引导性内容 |
| L2-A | 架构设计 | 核心系统架构、设计模式、状态管理 | 深度解析，建立完整知识框架 |
| L2-S | 安全系统 | 权限、沙箱、认证、安全机制 | 安全专题，集中讲解 |
| L2-E | 扩展机制 | MCP、Skills、Hooks、Plugins | 扩展开发专题 |
| L2-P | API服务 | API客户端、OAuth、流式、遥测 | 服务层专题 |
| L2-U | UI交互 | 渲染、输入、键绑定、主题 | 前端交互专题 |
| L2-F | 核心流程 | clear/compact/resume/rewind | 核心命令流程 |
| L3-T | 工具详解 | 单个工具完整实现 | 工具专项深度分析 |
| L3-C | 命令详解 | 单个命令完整实现 | 命令专项深度分析 |
| L3-X | 系统组件 | 系统级组件（Memory、配置等） | 系统组件专题 |
| L3-H | 高级主题 | Token优化、提示词、并行处理 | 进阶技术专题 |
| L3-I | 集成扩展 | LSP、Git、Chrome、Desktop集成 | 第三方集成专题 |
| L3-D | 设计模式 | 代码设计模式总结 | 模式抽象总结 |
