## 1. Claude Code 简介

### 1.1 什么是 Claude Code

Claude Code 是 Anthropic 官方推出的 AI 编程助手命令行工具，基于 Claude 模型提供：

- **代码生成与编辑** - 自然语言描述需求，AI 生成代码
- **代码理解与审查** - 分析代码库，提供审查建议
- **命令执行** - 在用户授权下执行 Shell 命令
- **多代理协作** - 复杂任务可拆分子任务并行执行

### 1.2 核心能力

| 能力 | 说明 |
|:---|:---|
| **工具调用** | Bash、FileEdit、FileRead 等 9 个核心工具 |
| **斜杠命令** | /commit、/review、/clear 等 78 个内置命令 |
| **MCP 扩展** | 通过 Model Context Protocol 集成外部工具 |
| **Skills 系统** | 可定制的能力模板 |
| **会话持久化** | 支持会话恢复和分支 |

## 2. 安装步骤

### 2.1 系统要求

| 系统 | 要求 |
|:---|:---|
| macOS | 10.15+ |
| Linux | glibc 2.17+ |
| Windows | WSL2 推荐 |

### 2.2 安装方式

**方式一：npm 安装（推荐）**

```bash
npm install -g @anthropic-ai/claude-code
```

**方式二：Homebrew 安装**

```bash
brew install claude-code
```

### 2.3 认证配置

首次运行需要配置 API Key：

```bash
claude
# 按提示完成 OAuth 认证
```

认证信息存储位置：
- macOS: Keychain
- Linux: libsecret（需安装）
- Windows: 明文存储（不推荐生产环境）

> **认证详解:** 见 [38-OAuth认证系统](./38-OAuth认证系统.md)

## 3. 基础使用

### 3.1 启动 CLI

```bash
# 在项目目录启动
cd your-project
claude

# 或指定工作目录
claude --cwd /path/to/project
```

### 3.2 交互模式

启动后进入交互式对话界面：

```
╭───────────────────────────────────────────────────────────╮
│ Claude Code                                               │
│ Model: claude-sonnet-4-6 | Mode: default                  │
╰───────────────────────────────────────────────────────────╯

> _
```

### 3.3 基本操作

| 操作 | 说明 |
|:---|:---|
| 输入文本 | 与 Claude 对话 |
| `/command` | 执行斜杠命令 |
| `Ctrl+C` | 中断当前操作 |
| `Ctrl+D` | 退出 CLI |
| `↑/↓` | 浏览历史输入 |

## 4. 第一个对话示例

### 4.1 简单问答

```
> 请解释这个项目的主要功能

Claude 会分析项目结构和代码，给出解释。

```

