# 桌面应用集成

**分析日期:** 2026-04-02

## 1. 桌面集成概述

Claude Code CLI 提供了两套桌面集成机制：

### 1.1 CLI → Desktop 会话移交

允许用户将当前 CLI 会话无缝转移到 Claude Desktop 应用继续操作。

**核心文件：**
- `commands/desktop/index.ts` - 命令注册入口
- `commands/desktop/desktop.tsx` - 命令实现
- `components/DesktopHandoff.tsx` - UI 交互组件
- `utils/desktopDeepLink.ts` - Desktop 深度链接工具

**工作流程：**
1. 用户执行 `/desktop` 或 `/app` 命令
2. 检测 Desktop 安装状态和版本兼容性
3. 刷新会话存储确保数据完整
4. 构建 `claude://resume?session=...&cwd=...` 深度链接
5. 打开 Desktop 应用，CLI 优雅退出

### 1.2 浏览器/外部 → CLI 深度链接

允许从浏览器或其他应用点击链接直接启动 CLI 会话。

**核心文件：**
- `utils/deepLink/protocolHandler.ts` - 协议处理入口
- `utils/deepLink/parseDeepLink.ts` - URI 解析器
- `utils/deepLink/registerProtocol.ts` - 协议注册
- `utils/deepLink/terminalLauncher.ts` - 终端启动器
- `utils/deepLink/banner.ts` - 安全警告提示
- `utils/deepLink/terminalPreference.ts` - 终端偏好记录

**工作流程：**
1. OS 注册 `claude-cli://` 协议
2. 用户点击 `claude-cli://open?q=...&repo=...` 链接
3. OS 调用 `claude --handle-uri <url>`
4. 解析 URI，检测用户终端
5. 在新终端窗口启动 Claude CLI

```typescript
// components/DesktopHandoff.tsx

useEffect(() => {
  async function performHandoff(): Promise<void> {
    setState('checking')
    const installStatus = await getDesktopInstallStatus()
    
    // 未安装 → 提示下载
    if (installStatus.status === 'not-installed') {
      setDownloadMessage('Claude Desktop is not installed.')
      setState('prompt-download')
      return
    }
    
    // 版本过旧 → 提示更新
    if (installStatus.status === 'version-too-old') {
      setDownloadMessage(`Claude Desktop needs to be updated (found v${installStatus.version}, need v1.1.2396+).`)
      setState('prompt-download')
      return
    }
    
    // 刷新会话存储
    setState('flushing')
    await flushSessionStorage()
    
    // 打开 Desktop
    setState('opening')
    const result = await openCurrentSessionInDesktop()
    
    if (!result.success) {
      setError(result.error ?? 'Failed to open Claude Desktop')
      setState('error')
      return
    }
    
    // 成功 → 延迟退出（让用户看到成功消息）
    setState('success')
    setTimeout(async (onDone) => {
      onDone('Session transferred to Claude Desktop', { display: 'system' })
      await gracefulShutdown(0, 'other')
    }, 500, onDone)
  }
  
  performHandoff().catch(err => {
    setError(errorMessage(err))
    setState('error')
  })
}, [onDone])
```

```typescript
async function isDesktopInstalled(): Promise<boolean> {
  if (isDevMode()) return true  // 开发模式假设已安装
  
  const platform = process.platform
  
  if (platform === 'darwin') {
    // macOS: 检查 /Applications/Claude.app
    return pathExists('/Applications/Claude.app')
  } else if (platform === 'linux') {
    // Linux: xdg-mime 查询协议处理器
    const { code, stdout } = await execFileNoThrow('xdg-mime', [
      'query', 'default', 'x-scheme-handler/claude'
    ])
    return code === 0 && stdout.trim().length > 0
  } else if (platform === 'win32') {
    // Windows: 注册表查询
    const { code } = await execFileNoThrow('reg', [
      'query', 'HKEY_CLASSES_ROOT\\claude', '/ve'
    ])
    return code === 0
  }
  
  return false
}
```

### 4.2 Desktop 版本检测

```typescript
async function getDesktopVersion(): Promise<string | null> {
  const platform = process.platform
  
  if (platform === 'darwin') {
    // macOS: 读取 Info.plist 的 CFBundleShortVersionString
    const { code, stdout } = await execFileNoThrow('defaults', [
      'read',
      '/Applications/Claude.app/Contents/Info.plist',
      'CFBundleShortVersionString'
    ])
    if (code !== 0) return null
    const version = stdout.trim()
    return version.length > 0 ? version : null
  } else if (platform === 'win32') {
    // Windows: 查找 %LOCALAPPDATA%\AnthropicClaude\app-X.Y.Z 目录
    const localAppData = process.env.LOCALAPPDATA
    if (!localAppData) return null
    const installDir = join(localAppData, 'AnthropicClaude')
    
    const entries = await readdir(installDir)
    const versions = entries
      .filter(e => e.startsWith('app-'))
      .map(e => e.slice(4))
      .filter(v => semverCoerce(v) !== null)
      .sort((a, b) => semverCoerce(a)!.compare(semverCoerce(b)!))
    
    return versions.length > 0 ? versions[versions.length - 1]! : null
  }
  
  return null
}
```

### 4.3 安装状态类型

```typescript
export type DesktopInstallStatus =
  | { status: 'not-installed' }
  | { status: 'version-too-old'; version: string }
  | { status: 'ready'; version: string }

export async function getDesktopInstallStatus(): Promise<DesktopInstallStatus> {
  const installed = await isDesktopInstalled()
  if (!installed) return { status: 'not-installed' }
  
  let version: string | null
  try {
    version = await getDesktopVersion()
  } catch {
    return { status: 'ready', version: 'unknown' }
  }
  
  if (!version) return { status: 'ready', version: 'unknown' }
  
  const coerced = semverCoerce(version)
  if (!coerced || !semverGte(coerced.version, MIN_DESKTOP_VERSION)) {
    return { status: 'version-too-old', version }
  }
  
  return { status: 'ready', version }
}
```

```typescript
async function openDeepLink(deepLinkUrl: string): Promise<boolean> {
  const platform = process.platform
  logForDebugging(`Opening deep link: ${deepLinkUrl}`)
  
  if (platform === 'darwin') {
    if (isDevMode()) {
      // 开发模式: AppleScript 路由到已运行的 Electron
      const { code } = await execFileNoThrow('osascript', [
        '-e', `tell application "Electron" to open location "${deepLinkUrl}"`
      ])
      return code === 0
    }
    // 生产环境: 标准 open 命令
    const { code } = await execFileNoThrow('open', [deepLinkUrl])
    return code === 0
  } else if (platform === 'linux') {
    const { code } = await execFileNoThrow('xdg-open', [deepLinkUrl])
    return code === 0
  } else if (platform === 'win32') {
    // Windows: cmd /c start 打开 URL
    const { code } = await execFileNoThrow('cmd', [
      '/c', 'start', '', deepLinkUrl
    ])
    return code === 0
  }
  
  return false
}
```

## 5. CLI Deep Link 协议处理

### 5.1 协议格式 (`claude-cli://`)

**基础格式：**
```
claude-cli://open?<params>
```

**支持的参数：**
- `q` - 预填充的提示内容（可选）
- `cwd` - 工作目录绝对路径（可选）
- `repo` - GitHub repo slug `owner/name`（可选）

**示例：**
```
claude-cli://open
claude-cli://open?q=hello+world
claude-cli://open?q=fix+tests&repo=anthropics/claude-code
claude-cli://open?cwd=/Users/dev/project
```

### 5.2 URI 解析器 (`parseDeepLink.ts`)

**安全验证流程：**

```typescript
export function parseDeepLink(uri: string): DeepLinkAction {
  // 1. 协议验证
  const normalized = uri.startsWith(`${DEEP_LINK_PROTOCOL}://`)
    ? uri
    : uri.startsWith(`${DEEP_LINK_PROTOCOL}:`)
      ? uri.replace(`${DEEP_LINK_PROTOCOL}:`, `${DEEP_LINK_PROTOCOL}://`)
      : null
  
  if (!normalized) {
    throw new Error(`Invalid deep link: expected ${DEEP_LINK_PROTOCOL}:// scheme, got "${uri}"`)
  }
  
  // 2. URL 解析
  let url: URL
  try {
    url = new URL(normalized)
  } catch {
    throw new Error(`Invalid deep link URL: "${uri}"`)
  }
  
  // 3. hostname 验证（必须是 "open"）
  if (url.hostname !== 'open') {
    throw new Error(`Unknown deep link action: "${url.hostname}"`)
  }
  
  // 4. cwd 路径验证
  const cwd = url.searchParams.get('cwd') ?? undefined
  if (cwd && !cwd.startsWith('/') && !/^[a-zA-Z]:[/\\]/.test(cwd)) {
    throw new Error(`Invalid cwd in deep link: must be an absolute path, got "${cwd}"`)
  }
  if (cwd && containsControlChars(cwd)) {
    throw new Error('Deep link cwd contains disallowed control characters')
  }
  if (cwd && cwd.length > MAX_CWD_LENGTH) {  // 4096
    throw new Error(`Deep link cwd exceeds ${MAX_CWD_LENGTH} characters`)
  }
  
  // 5. repo slug 格式验证
  const repo = url.searchParams.get('repo') ?? undefined
  if (repo && !REPO_SLUG_PATTERN.test(repo)) {  // /^[\w.-]+\/[\w.-]+$/
    throw new Error(`Invalid repo in deep link: expected "owner/repo", got "${repo}"`)
  }
  
  // 6. query 安全处理
  let query: string | undefined
  if (rawQuery && rawQuery.trim().length > 0) {
    // Unicode 清理（防止隐藏字符注入）
    query = partiallySanitizeUnicode(rawQuery.trim())
    if (containsControlChars(query)) {
      throw new Error('Deep link query contains disallowed control characters')
    }
    if (query.length > MAX_QUERY_LENGTH) {  // 5000
      throw new Error(`Deep link query exceeds ${MAX_QUERY_LENGTH} characters`)
    }
  }
  
  return { query, cwd, repo }
}
```

**控制字符检测：**
```typescript
function containsControlChars(s: string): boolean {
  for (let i = 0; i < s.length; i++) {
    const code = s.charCodeAt(i)
    if (code <= 0x1f || code === 0x7f) {  // ASCII 控制字符
      return true
    }
  }
  return false
}
```

**长度限制：**
```typescript
const MAX_QUERY_LENGTH = 5000   // 防止超长提示隐藏恶意指令
const MAX_CWD_LENGTH = 4096     // PATH_MAX 限制
```

### 5.4 协议处理器入口 (`protocolHandler.ts`)

**主入口函数：**

```typescript
export async function handleDeepLinkUri(uri: string): Promise<number> {
  logForDebugging(`Handling deep link URI: ${uri}`)
  
  // 1. 解析 URI
  let action
  try {
    action = parseDeepLink(uri)
  } catch (error) {
    console.error(`Deep link error: ${error.message}`)
    return 1
  }
  
  logForDebugging(`Parsed deep link action: ${jsonStringify(action)}`)
  
  // 2. 解析工作目录
  const { cwd, resolvedRepo } = await resolveCwd(action)
  
  // 3. 检查 git fetch 时间（显示 stale 提示）
  const lastFetch = resolvedRepo ? await readLastFetchTime(cwd) : undefined
  
  // 4. 在终端启动 Claude
  const launched = await launchInTerminal(process.execPath, {
    query: action.query,
    cwd,
    repo: resolvedRepo,
    lastFetchMs: lastFetch?.getTime(),
  })
  
  if (!launched) {
    console.error('Failed to open a terminal. Make sure a supported terminal emulator is installed.')
    return 1
  }
  
  return 0
}
```

**工作目录解析优先级：**

```typescript
async function resolveCwd(action: { cwd?: string; repo?: string }): Promise<{ cwd: string; resolvedRepo?: string }> {
  // 1. 显式 cwd 参数
  if (action.cwd) {
    return { cwd: action.cwd }
  }
  
  // 2. repo slug → githubRepoPaths 配置查询
  if (action.repo) {
    const known = getKnownPathsForRepo(action.repo)  // 从配置获取已知路径
    const existing = await filterExistingPaths(known) // 过滤存在的路径
    if (existing[0]) {
      logForDebugging(`Resolved repo ${action.repo} → ${existing[0]}`)
      return { cwd: existing[0], resolvedRepo: action.repo }
    }
    logForDebugging(`No local clone found for repo ${action.repo}, falling back to home`)
  }
  
  // 3. 默认：用户主目录
  return { cwd: homedir() }
}
```

```typescript
async function detectMacosTerminal(): Promise<TerminalInfo> {
  // 1. 存储的偏好（从之前交互会话保存）
  const stored = getGlobalConfig().deepLinkTerminal
  if (stored) {
    const match = MACOS_TERMINALS.find(t => t.app === stored)
    if (match) return { name: match.name, command: match.app }
  }
  
  // 2. TERM_PROGRAM 环境变量
  const termProgram = process.env.TERM_PROGRAM
  if (termProgram) {
    const normalized = termProgram.replace(/\.app$/i, '').toLowerCase()
    const match = MACOS_TERMINALS.find(
      t => t.app.toLowerCase() === normalized || t.name.toLowerCase() === normalized
    )
    if (match) return { name: match.name, command: match.app }

  // 3. Spotlight 搜索安装的应用
  for (const terminal of MACOS_TERMINALS) {
    const { code, stdout } = await execFileNoThrow('mdfind', [
      `kMDItemCFBundleIdentifier == "${terminal.bundleId}"`
    ], { timeout: 5000 })
    if (code === 0 && stdout.trim().length > 0) {
      return { name: terminal.name, command: terminal.app }
    }

  // 4. 直接检查 /Applications
  for (const terminal of MACOS_TERMINALS) {
    const { code } = await execFileNoThrow('ls', [`/Applications/${terminal.app}.app`])
    if (code === 0) return { name: terminal.name, command: terminal.app }
  }
  
  // 5. 默认 Terminal.app
  return { name: 'Terminal.app', command: 'Terminal' }
}
```

```typescript
  if (cwd) args.push(`--working-directory=${cwd}`)
  args.push('-e', claudePath, ...claudeArgs)  // 直接传递 argv

}

case 'Windows Terminal': {
  if (cwd) args.push('-d', cwd)
  args.push('--', claudePath, ...claudeArgs)  // 直接传递 argv
  return spawnDetached(terminal.command, args)
}
```

**Shell 转义函数：**

```typescript
// POSIX 单引号转义（iTerm/Terminal.app）
function shellQuote(s: string): string {
  return `'${s.replace(/'/g, "'\\''")}'`
}

// AppleScript 字符串转义
function appleScriptQuote(s: string): string {
  return `"${s.replace(/\\/g, '\\\\').replace(/"/g, '\\"')}"`
}

// PowerShell 单引号转义（唯一特殊序列是 ''）
function psQuote(s: string): string {
  return `'${s.replace(/'/g, "''")}'`
}

// cmd.exe 转义（不使用 CommandLineToArgvW 规则）
function cmdQuote(arg: string): string {
  const stripped = arg.replace(/"/g, '').replace(/%/g, '%%')  // 移除双引号，转义 %
  const escaped = stripped.replace(/(\\+)$/, '$1$1')  // 尾部反斜杠加倍
  return `"${escaped}"`
}
```

## 6. 安全警告 Banner

### 6.1 设计目的

浏览器和 Linux xdg-open 可能无确认直接打开深度链接，用户需要：
- 知道会话来自外部来源
- 检查工作目录（影响加载的 CLAUDE.md）
- 仔细审查预填充的提示

### 6.2 Banner 构建

```typescript
// utils/deepLink/banner.ts

export function buildDeepLinkBanner(info: DeepLinkBannerInfo): string {
  const lines = [
    `This session was opened by an external deep link in ${tildify(info.cwd)}`,
  ]
  
  // 显示 repo 解析信息
  if (info.repo) {
    const age = info.lastFetch ? formatRelativeTimeAgo(info.lastFetch) : 'never'
    const stale = !info.lastFetch || Date.now() - info.lastFetch.getTime() > STALE_FETCH_WARN_MS
    lines.push(
      `Resolved ${info.repo} from local clones · last fetched ${age}${stale ? ' — CLAUDE.md may be stale' : ''}`,
    )
  }
  
  // 显示预填充提示警告
  if (info.prefillLength) {
    lines.push(
      info.prefillLength > LONG_PREFILL_THRESHOLD  // 1000 chars
        ? `The prompt below (${formatNumber(info.prefillLength)} chars) was supplied by the link — scroll to review the entire prompt before pressing Enter.`
        : 'The prompt below was supplied by the link — review carefully before pressing Enter.',
    )
  }
  
  return lines.join('\n')
}
```

### 6.3 Git Fetch 时间检查

```typescript
export async function readLastFetchTime(cwd: string): Promise<Date | undefined> {
  const gitDir = await getGitDir(cwd)
  if (!gitDir) return undefined
  
  const commonDir = await getCommonDir(gitDir)
  
  // 检查 worktree 的两个位置
  const [local, common] = await Promise.all([
    mtimeOrUndefined(join(gitDir, 'FETCH_HEAD')),
    commonDir ? mtimeOrUndefined(join(commonDir, 'FETCH_HEAD')) : Promise.resolve(undefined),
  ])
  
  if (local && common) return local > common ? local : common
  return local ?? common
}
```
